# Blog Project 📝

Blog simples desenvolvido para praticar **front-end**, estrutura de conteúdo e interações com o usuário.  
Visual moderno com cards responsivos, hover elegante e layout intuitivo.

---

## 📚 Tecnologias

- HTML5
- CSS3 (estilizado com gradientes, sombras e transições)
- JavaScript (DOM, filtragem de posts, localStorage)
- Git/GitHub

---

## 📂 Estrutura do projeto

```text
blog-project/
 ├── index.html        # Página principal do blog
 ├── post.html         # Página individual de cada post
 ├── css/blog.css      # Estilo atualizado e moderno
 ├── JS/Blog.js        # Lógica de exibição e filtragem de posts
 ├── JS/post.js        # Lógica de exibição do post individual
 └── imagens/          # Imagens dos posts
